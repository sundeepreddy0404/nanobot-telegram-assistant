## ✨ Overview

Telegram Nanobot AI is a lightweight AI assistant built using:

- 🧠 Nanobot Agent Framework  
- 🤖 Telegram Bot API  
- 🌐 OpenRouter LLM Gateway  

It enables real-time AI conversations directly inside Telegram.

---

## ⚡ Key Features

- 🚀 Ultra-Lightweight Agent Architecture  
- 🧠 LLM-powered responses  
- 🔌 Easy Telegram integration  
- 🔐 Secure environment-based configuration  
- 🛠 Extendable tool + memory support
